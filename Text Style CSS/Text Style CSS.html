Untuk menetukan jarak antara satu elemen dengan elemen yang lain dapat menggunakan margin, sedangakan untuk mengatur jarak antara border elemen dengan konten di dalamnya menggunakan padding. Font Style

CSS dapat mengatur style dari text yang ada pada halaman web ayng dibuat. Terdapat beberapa property yang dapat digunakan untuk mengatur style text yaitu sebagai berikut.

  • text-align digunakan untuk mengatur text, value yang dapat digunakan yaitu left,right, dan center
  • color digunakan untuk memberikan warna pada text
  • letter-spacing digunakan untuk mengatur jarak antar karakter
  • word-spacing digunakan untuk mengatur jarak antar kata
